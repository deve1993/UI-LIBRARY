# FeaturesTabs

Tabbed interface for showcasing features interactively.

## Usage

```tsx
import { FeaturesTabs } from '@/components/sections/FeaturesTabs';

const tabs = [
  {
    id: '1',
    label: 'Feature 1',
    title: 'Amazing Feature',
    description: 'Description here',
  },
];

<FeaturesTabs tabs={tabs} />
```
