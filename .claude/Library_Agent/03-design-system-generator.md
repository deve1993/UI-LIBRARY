```javascript
module.exports = {
  theme: {
    extend: {
      colors: tokens.colors,
      fontFamily: tokens.typography.fontFamily,
      fontSize: tokens.typography.fontSize,
      spacing: tokens.spacing,
      borderRadius: tokens.borderRadius,
      boxShadow: tokens.shadows,
      screens: breakpoints,
    },
  },
}
```

## Best Practices

- **Consistenza**: Usa sempre multipli base (4px spacing, 8px per breakpoints)
- **AccessibilitÃ **: Contrasta colori WCAG AA minimum (4.5:1 per testo)
- **ScalabilitÃ **: Design tokens facilitano temi multipli
- **Type-safety**: Export types per TypeScript
- **Documentation**: Commenti chiari per ogni token
